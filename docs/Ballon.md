`Ballon` (component)
====================

a component to show text inside a ballon

Props
-----

### `color`

background color of the ballon

type: `string`
defaultValue: `"#f3f"`


### `containerStyle` (required)

the style for the container view

type: `ViewStyleProp`


### `textStyle` (required)

the style for the TextInput inside ballon

type: `TextStyleProp`
